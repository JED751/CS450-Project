# Netflix Data Visualization Dashboard

This project is an interactive data visualization dashboard built using **React.js** and **D3.js**.  
It explores trends and patterns in Netflixâ€™s catalog of both movies and TV shows.

## Live Demo
https://jed751.github.io/CS450-Project/

## Dataset
Netflix Movies and TV Shows Dataset (Kaggle)  
https://www.kaggle.com/datasets/shivamb/netflix-shows

## Features
- Titles added per year (line chart)
- Genre distribution (horizontal bar chart)
- Country distribution (bar chart)
- Duration / season distribution (histogram)
- Interactive filtering by content type (Movie / TV Show / All)
